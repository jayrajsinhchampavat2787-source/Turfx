Turfx is a web-based cricket turf booking platform that allows users to
check availability and book turf slots online easily.

## Features
- User Registration & Login
- Turf Slot Booking System
- Real-time Slot Availability
- Booking Confirmation
- Admin Panel for Managing Slots

## Tech Stack
- Backend: Django
- Frontend: HTML, CSS
- Database: SQLite
- Authentication: Django Auth

## How to Run Locally

1. Clone the repository
   git clone https://github.com/jayrajsinhchampavat2787-source/Turfx

2. Go to project directory
   cd Turfx

3. Install dependencies
   pip install -r requirements.txt

4. Run the development server
   python manage.py runserver
